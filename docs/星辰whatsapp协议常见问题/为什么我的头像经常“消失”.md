# 为什么我的头像经常“消失”

分类：星辰whatsapp协议常见问题
更新时间：2026-03-29T08:44:49.898Z

## 消失的原因：

号跟星辰有交互就可能会有临时头像 ，如果是登陆前这个号跟星辰上的号有交互 缓存到他头像了 你登陆的时候看到有就没设置了 但是这种方式获取的头像是好友的头像有有效期

头像属于星辰上的共享资源 所以会实际没有 但是你还看得到头像

过期就变成消失了

## 怎么避免问题出现：

星辰跟手机一样每次登陆都要设置头像，每次登录后，不管有没有头像，最好重新设置一下头像

方式1、上号后，点击编辑头像，上传头像，点击确认按钮

![](../../site/assets/images/EFK4RD4xAv-c0d74114b0.png)

方式2、坐席找到账号，右键点击选择修改资料

![](../../site/assets/images/yfIaBXPHhk-a9ee5a378b.png)

弹出修改弹窗，点击头像的按钮，选择图片，就可以修改头像

![](../../site/assets/images/2nr8_b1aNz-f759df22fc.png)
