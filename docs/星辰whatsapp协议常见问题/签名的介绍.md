# 签名的介绍

分类：星辰whatsapp协议常见问题
更新时间：2026-03-31T07:01:12.703Z

在后管设置签名：打开WhatsApp列表，点击编辑签名，弹出弹窗，可以编辑签名，设置时间，点击确认按钮

![](../../site/assets/images/00-Q4__ldx-c468a1edd9.png)

签名设置成功，且有倒计时

![](../../site/assets/images/YUjwMXQ0kY-564face54a.png)

在坐席设置签名：找到账号，右键点击修改资料按钮

![](../../site/assets/images/CheHjt8G_D-00a8fa2357.png)

弹出弹窗，点击设置状态按钮

![](../../site/assets/images/jb4q3ZDS68-924d7ba60a.png)

输入状态，选择时间，点击保存按钮

![](../../site/assets/images/CzIWTEG0g0-5ea58bcfce.png)

设置成功

![](../../site/assets/images/1kmhf85z3T-a475c95616.png)

**同个账号后管和坐席的签名状态是同步的**

**签名过期的含义
比如你签名设置了24小时的笑脸在线 24小时01分就是已过期 别人再看你的签名就是默认签名**

放在不动有没有关系
