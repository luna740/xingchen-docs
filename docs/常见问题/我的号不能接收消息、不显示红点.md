# 我的号不能接收消息、不显示红点

分类：常见问题
更新时间：2026-05-23T00:00:00+08:00
ID：8cc0f4046316564933af65a8

**如果号码不能接收消息，或者新消息不显示红点提醒，通常可以通过让号码离线后再重新上线来恢复。**

> 处理方式：对该号码执行【离线】操作，等待状态变为离线后，再执行【上线】。

## 一、适用情况

出现以下情况时，可以先按此方式处理：

1. 号码收不到新消息。
2. 有新消息但列表不显示红点。
3. 消息提醒状态看起来没有正常刷新。

## 二、处理方法

1. 找到对应 WhatsApp 号码。
2. 对该号码执行【离线】。
3. 等待号码状态变为离线。
4. 再对该号码执行【上线】。
5. 上线完成后，重新查看消息接收和红点提醒是否恢复。

> 提示：此类问题优先执行号码离线再上线，不需要重复刷新页面或反复切换会话。
