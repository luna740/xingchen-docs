# 我的号不能接收消息、不显示红点

分类：常见问题
更新时间：2026-05-26T00:00:00+08:00
ID：8cc0f4046316564933af65a8

号码不能接收消息、不显示红点时，先按下面顺序判断原因。

## 一、本地网络波动

如果本地网络波动，可能会导致星辰客户端和星辰服务器短暂断线，页面上的消息状态没有及时刷新。

处理方式：先刷新浏览器，再重新查看消息接收和红点提醒是否恢复。

## 二、账号绑定代理 IP 波动

如果账号绑定的代理 IP 波动，也可能影响消息接收或提醒状态。

这种情况星辰会自动尝试执行修复，可以先等待系统自动处理。

## 三、区分不能接收和不能解密

如果刷新浏览器和自动修复后仍没有解决，请先判断问题到底是：

1. 不能接收消息。
2. 能收到消息，但不能解密，表现为显示延迟、等候等提示。

如果是消息解密失败，请准确提交号码、截图和问题描述。截图必须能看出问题发生在哪个联系人或哪个群组，问题描述请明确写为：

> 不能解密接收到的信息

## 四、等待人工客服复核

如果不是本地网络波动、代理 IP 波动，也不是消息解密失败，请等待人工客服进一步复核。
