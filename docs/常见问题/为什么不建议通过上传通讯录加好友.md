# 为什么不建议通过上传通讯录加好友

分类：常见问题
更新时间：2026-05-20T22:56:33+08:00
ID：63e2871f32f32b3994cef5d6

**因为盾牌超能力的原因，业务操作并不需要加好友。上传通讯录批量加好友会增加账号风控风险，不建议作为常规操作。**

关于加好友风险，请先参考：[为什么不建议加好友](./为什么不建议加好友.md)

## 一、为什么不建议上传通讯录加好友

上传通讯录的本质是批量导入联系人并尝试建立好友关系。该操作容易被官方识别为异常加好友行为，尤其在短时间内批量处理时，风险更高。

主要风险：

1. 批量加好友容易触发官方风控。
2. 账号可能被限制、掉线或封禁。
3. 加好友并不是聊天、建群、邀请进群的必要条件。
4. 业务上已有盾牌能力时，通常可以直接聊天，不需要额外添加好友。

## 二、业务应该怎么做

1. 优先通过正常对话建立联系。
2. 对方有盾牌时，可以直接主动聊天。
3. 建群或邀请进群不依赖加好友。
4. 如果必须导入联系人，请先确认业务必要性，并控制数量和频率。

## 三、上传通讯录入口说明

系统中仍保留上传通讯录入口，但不建议将其作为常规加好友方式。

入口位置：

1. 找到在线账号。
2. 在账号列表页面点击【上传通讯录】。
3. 系统会弹出上传通讯录窗口。
