# 提问模板和星辰助手上线

分类：常见问题
更新时间：2026-05-26T00:00:00+08:00
ID：a2980b63aff5e5944f3a8147

星辰助手已上线，所有问题请使用 `/q` 按固定格式提问。

## 一、聊天相关问题

对聊天相关的问题进行提问时，请使用：

```text
/q 号码 图片 问题描述
```

聊天相关问题需要带上号码、图片和问题描述，方便确认问题发生在哪个号码、哪个会话或哪个聊天页面。

## 二、涉及号码的非聊天问题

对涉及号码但不是聊天相关的问题进行提问时，请使用：

```text
/q 号码 问题描述
```

例如号码状态、登录、封号、在线状态等问题，可以按这个格式提交。

## 三、不涉及号码的问题

对不涉及具体号码的问题进行提问时，请使用：

```text
/q 问题描述
```

例如：

```text
/q 怎么打开静音通知
```

## 四、人工客服处理时间和记录要求

星辰人工客服上班时间为北京时间每周日到周五，早上 9 点到晚上 9 点。

涉及号码的聊天问题，大部分需要人工查证。提交这类问题时，必须保证问题按格式发送，并且包含号码、截图和问题描述。截图需要能看出问题发生在哪个联系人或哪个群，问题描述越详细，星辰助手能定位到自助处理的概率越高。

如果聊天问题不含号码，或者截图不符合格式，星辰助手会提醒补充信息。你需要按完整格式重新发送一次问题，问题才会被记录到机器人里，再等待人工客服上班后处理。
