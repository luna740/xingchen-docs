# 提问模板和星辰助手上线

分类：常见问题
更新时间：2026-05-26T00:00:00+08:00
ID：a2980b63aff5e5944f3a8147

星辰助手已上线，提问统一使用 `/q`。

## 一、按问题类型选择格式

聊天相关问题：

```text
/q 号码 图片 问题描述
```

涉及号码但不是聊天的问题：

```text
/q 号码 问题描述
```

不涉及具体号码的问题：

```text
/q 问题描述
```

例如：

```text
/q 怎么打开静音通知
```

## 二、聊天问题提交要求

涉及聊天的问题大部分需要人工查证，请一次性提交完整信息：号码、截图和问题描述。

截图需要能看出问题发生在哪个联系人或哪个群；问题描述越清楚，星辰助手越容易判断是否可以自助处理。

如果没有号码，或者截图看不出问题发生位置，星辰助手会提醒补充信息。请按完整格式重新发送一次，问题才会被记录到机器人里。

## 三、人工客服时间

星辰人工客服上班时间为北京时间每周日到周五，早上 9 点到晚上 9 点。

需要人工查证的问题，会在记录后等待人工客服上班处理。
