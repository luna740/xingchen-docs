# 为什么不建议加好友

分类：常见问题
更新时间：2026-05-20T22:44:03+08:00
ID：07e22c3e84fe58064e09a284

**本文说明为什么不建议在星辰中频繁点击【加好友】，以及有盾牌和无盾牌账号在页面上的区别。**

> 注意：不加好友不影响聊天、建群、邀请进群等正常功能。为了降低账号风控风险，建议不要主动点击【加好友】。

## 一、为什么不建议加好友

根据当前统计，不管是主动添加好友，还是被动添加好友，都可能被官方记录并触发风控。

加好友可能带来的风险：

1. 增加账号被官方风控的概率。
2. 增加账号被限制或封禁的风险。
3. 大量、频繁点击【加好友】更容易被识别为异常操作。

## 二、什么情况下会显示加好友按钮

只有带有信任盾牌的账号，页面才会显示【添加好友】按钮。没有信任盾牌的账号，不会显示该按钮。

有信任盾牌时，页面会显示添加好友入口：

![](../../site/assets/images/pXXBreV0yY-1030fe8236.png)

没有信任盾牌时，页面不会显示添加好友入口：

![](../../site/assets/images/MR0aO9KIo2-dcc96a89e8.png)

## 三、推荐怎么做

1. 不要为了使用功能去点击【加好友】。
2. 只要已经产生对话，并且对方有盾牌，就可以正常聊天。
3. 建群、邀请进群等操作不依赖加好友。
4. 如果业务必须操作，请控制频率，不要短时间大量点击。

> 建议：优先通过正常聊天、群聊、邀请进群完成业务，不要把【加好友】作为常规操作。
