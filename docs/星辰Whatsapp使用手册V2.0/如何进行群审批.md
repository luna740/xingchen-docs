# 如何进行群审批

分类：星辰Whatsapp使用手册V2.0
更新时间：2026-05-20T20:37:21+08:00
ID：75d4ce22d9f68f92fa0d7e6e

**本文说明收到进群申请后，如何进入群审批页面并处理审批。适用于通过邀请链接申请进群，或群内普通成员请求添加其他人员进群的场景。**

## 一、查看进群申请通知

1. 当有人通过邀请链接申请进群时，系统会产生通知消息。
2. 当群内普通成员请求添加其他人员进群时，也会产生通知消息。

   ![](../../site/assets/images/iT_YzsLtWD-ab2b1871db.png)

## 二、进入审批界面

1. 在通知消息中点击相关入口。
2. 页面会跳转到群审批界面。

   ![](../../site/assets/images/KceWFqfXFE-ca47e5a943.png)

## 三、查找并处理审批

1. 审批界面会按时间倒序展示申请记录，最新申请会排在前面。
2. 如果申请记录较多，可以使用模糊搜索快速查找目标申请。
3. 找到对应申请后，根据实际情况进行审批处理。

   ![](../../site/assets/images/20UEcxqUTD-4f249d7115.png)

## 四、审批频率注意事项

> 注意：审批操作过快可能会触发限制。建议控制审批频率，约 20 秒处理一个申请。
