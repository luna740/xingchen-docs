# 如何使用批量加群

分类：星辰Whatsapp使用手册V2.0
更新时间：2026-03-12T09:29:46.000Z

## 一般不会触发风控，如果你不确定适合的频率，不要改变默认的间隔配置

1、点击坐席右边最下面的“批量加群/退群”按钮

2、粘贴要添加的群链接，设置等待时间，点击“批量加群”，就开始批量加群了

![](../../site/assets/images/jrCia-Wub_-52d4e3bbb6.png)

3、开始加群后，每加一个群，成功失败都会有提示，也会显示加群记录

![](../../site/assets/images/DFt1oCiy7F-ee0d234743.png)

## 4、注：在批量加群中不能刷新浏览器，刷新会自动停止批量加群，同时也不能开启批量退群，也会停止批量加群，因为加群和退群操作只能一项进行，不能同时进行。
