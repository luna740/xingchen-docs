# 如何在星辰登录whatsapp账号

分类：星辰Whatsapp使用手册V2.0
更新时间：2026-05-12T01:28:39.054Z

**预计要登录多少个号，找阿宝开通试用端口，一个管理员最多可以使用100个端口一个月（登录100个试用号）**

## 购买IP：

点击左侧菜单【IP列表】，进入IP列表，IP列表可查看当前账号拥有的IP情况
点击右侧的【购买IP】按钮，触发【购买IP】弹窗

![](../../site/assets/images/zukmwa7IZW-ac64f475bc.png)

①下拉可选择要购买哪个国家的IP
②选择购买的IP数量（目前IP仅支持单次购买30天）
③选择'支付方式'（建议使用余额支付，方便后续自动续费）
④选择'自动续费'（为避免因过期影响账号的正常使用，建议勾选并保持账号余额充足）
⑤勾选同意《服务协议》并点击【前往支付】按钮，触发【确认订单】弹窗
⑥确认订单无误后，点击【确认支付】按钮
⑦弹出【支付成功】弹窗即购买端口成功
注意：IP购买需要一定的时间发货/到账，为保证正常使用，建议提前购买IP。

![](../../site/assets/images/ht7o55Co4x-7f8b61a70d.png)

![](../../site/assets/images/jXLbKgRBEt-982b576941.png)

![](../../site/assets/images/9OKAADOYLf-be6448c8b0.png)

## 将whats app账号登录到管理员账号下：

※绑定操作前建议先购买IP

1、点击扫码登录按钮，弹出登录弹窗

![](../../site/assets/images/8BrgAWN41H-aec3eeffd6.png)

2、添加好正确登录信息，点击确认按钮，弹出输入验证码弹窗，输入收到的验证码，点击确认按钮

![](../../site/assets/images/8_tBTq4P4H-024a697adc.png)

![](../../site/assets/images/u8Phpw7b7j-2ae6016992.png)

3、【WhatsApp账号】成功加入到【星辰客服系统】中，支持绑定坐席进行操作
