# 如何解封

分类：星辰Whatsapp使用手册V2.0
更新时间：2026-05-20T21:25:50+08:00
ID：d4b3c2a02ced51eec27b6ad2

**本文说明如何判断 WhatsApp 账号是否被封、如何提交解封申请，以及账号解封后如何重新登录。**

> 注意：账号进入【解封中】后不会自动变为【已解封】，需要手动点击【查看解封状态】查询结果。建议至少每半天查询一次，避免超过有效期。

## 一、判断账号是否被封

1. 查看坐席账号头像是否出现红点。
2. 将鼠标移动到红点上，查看是否提示【封禁】。
3. 如果账号无法正常私聊或群聊，同时出现封禁提示，说明该账号已被封禁。

   ![](../../site/assets/images/XnZfQLX0rG-be1025b1e1.png)

## 二、提交解封申请

1. 进入账号列表页面。
2. 将状态筛选为【封禁】，点击【搜索】。
3. 找到需要申请解封的账号。
4. 点击【申请解封】，打开申请解封弹窗。
5. 解封理由可以不填写，直接点击【确认】提交申请。

   ![](../../site/assets/images/h0uXb-AiAq-5599b899c9.png)

## 三、查看解封状态

1. 提交申请后，账号会进入【解封中】状态。
2. 【解封中】不会自动变为【已解封】，需要手动点击【查看解封状态】查询结果。
3. 建议至少每半天查询一次状态，确认账号是否已经完成解封。
4. 解封申请和解封状态查询都有有效期，请及时查询，避免超过有效期后无法继续查看或处理。

   ![](../../site/assets/images/E1XwzD8RG--200d95a916.png)

## 四、已解封后重新登录

> 注意：已解封账号需要手机插卡复接登录。

1. 在账号列表中筛选并找到【已解封】账号。
2. 点击【登录】，打开登录弹窗。
3. 检查登录信息是否正确。
4. 如果此前释放过 IP，需要重新选择 IP。
5. 确认无误后，点击【确认】完成重新登录。

   ![](../../site/assets/images/Bt0cKbErjH-c31d9af7ed.png)
