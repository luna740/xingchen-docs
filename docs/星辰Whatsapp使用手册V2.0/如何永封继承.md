# 如何永封继承

分类：星辰Whatsapp使用手册V2.0
更新时间：2026-05-20T21:32:02+08:00
ID：1bb06441c03c0b4cf92ddb34

**本文说明如何将【永久封禁】账号的会话记录继承到新的在线账号下，方便继续跟进原账号中的客户会话。**

> 注意：一个【永久封禁】账号只能被一个账号继承。如果继承错了，需要先找到永封的号，解除继承关系，再重新选择另外的账号去继承永封的号。
> 注意：一个账号允许继承多个【永封号】，也允许子孙继承；比如A继承了永封B号后，A也永封了，C继承A就会继承A和B粉丝的并集
> 注意：【封禁】、【解封中】、【已解封】、【登出】状态的号，可以在联系【F-对接群】里面反馈设置状态为【永久封禁】，该操作不可取消，慎重考虑

## 一、适用场景

当原账号已经处于【永久封禁】状态，但账号内仍有需要继续跟进的会话记录时，可以使用永封继承功能，将这些会话记录关联到新的账号下继续处理。

## 二、操作前确认

1. 原账号必须是【永久封禁】状态，并且账号下有会话记录。
2. 新账号需要先绑定坐席，否则不会出现继承永封相关功能。
3. 新账号必须处于【在线】状态，才可以作为继承目标账号。
4. 如果页面没有操作权限，请联系【F-对接群】里面反馈。

## 三、发起永封继承

1. 在账号列表中筛选【永久封禁】状态。
2. 找到需要继承会话记录的永久封禁账号，并复制该账号。

   ![](../../site/assets/images/bk3FCbvQzK-e8d80987c4.png)

3. 将账号状态筛选为【在线】。
4. 找到用于承接会话记录的新账号，点击【新增关联】。
5. 在弹窗中选择需要继承的永久封禁账号，也可以直接粘贴账号进行搜索。
6. 确认账号无误后，点击【确认】完成继承。

   > 提示：账号后面的蓝色数字表示相同粉丝数，可用于辅助判断继承对象是否正确。

   ![](../../site/assets/images/mpjTzb3Rle-53b82f0b5f.png)

## 四、查看继承效果

继承完成后，进入坐席查看会话列表。新账号下会显示继承后的会话数据，可继续跟进原永久封禁账号中的客户。

## 五、解除永封继承

如果继承账号选择错误，或需要重新调整继承关系，可以先解除当前继承。

1. 找到需要解除继承的账号。
2. 点击【关联账号数】下方的数字，进入关联列表。

   ![](../../site/assets/images/4aZ_GyevcT-95035b1565.png)

3. 在关联列表中找到需要解除的账号。
4. 点击【删除】，即可解除当前继承关系。

   ![](../../site/assets/images/53jFq1LrGp-6a5549252e.png)
