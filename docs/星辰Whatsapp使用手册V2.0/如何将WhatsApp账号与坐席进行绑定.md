# 如何将WhatsApp账号与坐席进行绑定

分类：星辰Whatsapp使用手册V2.0
更新时间：2026-04-02T06:12:42.355Z

“如果要更改坐席找风控”

## 新建座席账号

展开左侧菜单的【客服坐席】并点击【座席列表】按钮

![](../../site/assets/images/hDQb6UAHeJ-fc6d2060be.png)

进入【座席列表】后，点击【新增】按钮，触发【新增座席账号】弹窗

![](../../site/assets/images/ivDF8Eek4O-7c164b9b23.png)

![](../../site/assets/images/cunM36JQn7-b76b9026ba.png)

正确填写各项内容后，点击【确定】按钮，即可成功新建坐席账号

![](../../site/assets/images/GsAG5kF1D7-9348d11869.png)

## 将WhatsApp账号绑定到座席

点击座席账号旁的【编辑】按钮触发编辑弹窗

![](../../site/assets/images/edmLwJPj8j-9b5275d1a0.png)

点击【WhatsApp账号】旁的文本框，触发弹窗选择要绑定的WhatsApp账号

![](../../site/assets/images/e-N43l7OJa-5faf0eec6d.png)

勾选要与"座席账号"绑定的"WhatsApp账号"后，点击【确定】按钮

![](../../site/assets/images/GqDGLUdIRf-eb927d61e8.png)

再次点击【确定】应用"绑定关系

![](../../site/assets/images/EC-WAwgfI1-9860e0f427.png)

提示"编辑成功"且"绑定账号数"发生变化即绑定WhatsApp账号成功。通过【坐席系统】登录后，即可操作该WhatsApp账号

![](../../site/assets/images/qTN1gsZA6V-e655212994.png)

## 坐席功能权限介绍

1、好友权限，选中好友权限后，坐席系统就可以开启查找联系人功能，不选中这个权限，就没有查找联系人这个功能。

![](../../site/assets/images/FokS5hW5ip-1a38caa1c7.png)

![](../../site/assets/images/hUT11-PWuR-4d72132ee5.png)

2、群组/社群权限，选中权限后，坐席系统就可以使用创建群组和创建社群功能。不选中这个权限，就没有这个功能

![](../../site/assets/images/qU8ni7gCru-57529cdf9f.png)

![](../../site/assets/images/cT6DG-S15r-665d0450a2.png)

3、聊天权限，选中权限后，坐席系统就可以进行私聊和群聊的聊天功能。不选中这个权限，就没有这个功能

![](../../site/assets/images/NomAZA1Yav-24192bcf3d.png)

![](../../site/assets/images/f8SwCB7Szj-07e2032b10.png)
