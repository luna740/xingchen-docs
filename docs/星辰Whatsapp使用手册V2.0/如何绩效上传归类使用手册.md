## 操作步骤

1. 对方发的图片显示上传按钮后，可以点击按钮上传绩效。

   注意：发了图片后，需要回复信息，或者对方再发送除图片、视频之外的信息，才会显示上传按钮。对方发送图文加文字时，会直接显示上传按钮。

   ![](../../site/assets/images/manual-performance-upload-p1d_bqfn-n-523829ade3.png)

2. 填写上传信息，按上传表单填写正确信息，点击提交。

   ![](../../site/assets/images/manual-performance-upload-yu6zrwywkk-5ad2a6ade6.png)

3. 信息提交后会进入审核状态，后台审核后，可以查询状态。

   ![](../../site/assets/images/manual-performance-upload-mrltrsnskz-2a4fd06690.png)

   ![](../../site/assets/images/manual-performance-upload-irxxw6sicb-700afd6a7b.png)

4. 如果出现审核不通过，需要重新修改提交。

   ![](../../site/assets/images/manual-performance-upload-x0s7p6bwib-aef75184cd.png)

5. 等待出现审核通过，即表示数据上传正确并已审核通过。

   ![](../../site/assets/images/manual-performance-upload-20ksrgyoqb-3169368425.png)
