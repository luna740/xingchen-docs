# 全局翻译修改

分类：星辰Whatsapp使用手册V2.0
更新时间：2026-03-12T09:28:41.423Z

点击账号名称右边的展开按钮，再点击翻译设置，弹出设置界面

![](../../site/assets/images/sD-7tntSBR-21afd2e5ab.png)

根据自己要求修改设置：翻译引擎、API_KEY、消息翻译、发送翻译、群聊翻译，然后点击提交按钮就设置成功了

![](../../site/assets/images/LOlfoQjoju-530a310b95.png)

进入坐席界面，翻译设置勾选”启用默认配置“，就会使用全局配置了

![](../../site/assets/images/xAzyp4jnDI-642c9ebf27.png)
