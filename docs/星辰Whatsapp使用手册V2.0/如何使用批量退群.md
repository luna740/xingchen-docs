# 如何使用批量退群

分类：星辰Whatsapp使用手册V2.0
更新时间：2026-03-12T09:29:17.301Z

## 一般不会触发风控，如果你不确定适合的频率，不要改变默认的间隔配置

1、点击坐席右边最下面的“批量加群/退群”按钮

2、切换到批量退群页面，勾选要退出群聊，设置等待时间，点击批量退群，就开始退群操作了

![](../../site/assets/images/Dmot_oBNYi-b8c09ce333.png)

3、开始退群后，每退出一个群，成功失败都会有提示，也会显示退群记录，可以“点击停止”，停止退群操作

![](../../site/assets/images/DxQFspICid-f6d2d45bb6.png)

## 4、注：在批量退群中不能刷新浏览器，刷新会自动停止批量退群，同时也不能开启批量加群，也会停止批量退群，因为加群和退群操作只能一项进行，不能同时进行
