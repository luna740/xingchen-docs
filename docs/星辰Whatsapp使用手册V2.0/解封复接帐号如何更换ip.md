# 解封复接帐号如何更换ip

分类：星辰Whatsapp使用手册V2.0
更新时间：2026-03-16T09:11:13.111Z

## 换ip再上线 在线不允许更换ip

1、筛选"已解封"状态的账号，点击释放ip按钮，弹出确认弹窗，点击确认按钮ip释放成功

![](../../site/assets/images/-Y_6CeAotV-e605ac09b6.png)

![](../../site/assets/images/NUuWyc74AI-55691541ea.png)

2、点击登录按钮，弹窗登录弹窗，可以重新选择ip

![](../../site/assets/images/Xme1t3SQSz-aa4e723e4e.png)
