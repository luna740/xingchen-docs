# 如何一键已读

分类：星辰Whatsapp使用手册V2.0
更新时间：2026-05-20T21:59:53+08:00
ID：91607c6bee58eee082787eba

**本文说明如何使用【一键已读】功能，将账号中未归档的消息批量标记为已读。**

> 注意：【一键已读】会批量处理当前账号未归档的消息，请确认无需逐条查看后再操作。

## 一、进入账号列表

1. 在左侧菜单进入【账号管理】。
2. 打开对应的 WhatsApp 账号列表。
3. 找到需要执行一键已读的账号。

## 二、执行一键已读

1. 在账号列表中，对目标账号点击鼠标右键。
2. 在右键菜单中点击【一键已读】。
3. 系统会将该账号中未归档的消息批量标记为已读。

   ![](../../site/assets/images/Xx3AATUEMB-7e6cc4ef78.png)

## 三、使用建议

1. 不建议频繁点击【一键已读】，避免影响坐席正常判断新消息。
2. 如果账号仍在持续接收客户消息，建议先确认是否有重要消息需要人工查看。
3. 一键已读适合用于清理已确认无需处理的未读消息。
