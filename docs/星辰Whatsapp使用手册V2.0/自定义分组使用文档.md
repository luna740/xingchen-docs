# 自定义分组使用文档

分类：星辰Whatsapp使用手册V2.0
更新时间：2026-04-02T13:11:21.502Z

**一，创建分组**

1、点击管理分组按钮，

![](../../site/assets/images/1WEoTQdxtu-a2bd8d2f0e.png)

2、点击“+”按钮

![](../../site/assets/images/qTRNBHf9vE-6b284d77c9.png)

3、弹出创建界面后，填写分组名，勾选会话，点击“创建”按钮

![](../../site/assets/images/L3G6WHdEy3-99303b0e46.png)

4、提示创建分组成功，

![](../../site/assets/images/mK13phpu6g-903ad25c6d.png)

**二、分组操作**

**修改分组名称**

1、点击更改分组名称按钮

![](../../site/assets/images/gFICXMAkbO-68929f1bfb.png)

2、弹出弹窗后，填写新名称，点击确认按钮

![](../../site/assets/images/uQk-cYdwq_-b21b0c26f2.png)

**添加聊天**

1、点击添加聊天按钮

![](../../site/assets/images/0IdnEv-Y9T-542b0d9b8d.png)

2、勾选聊天会话，点击添加按钮

![](../../site/assets/images/PcN0krMZBF-5d9510c288.png)

3、添加成功，分组下看到勾选的聊天会话

![](../../site/assets/images/DB6fNxEjYx-6a0957cf8b.png)

**删除聊天**

1、点击移除聊天按钮

![](../../site/assets/images/N8b3Ni61LR-d1d6cc54d6.png)

2、弹出弹窗后，勾选想移除的聊天，点击移除按钮

![](../../site/assets/images/4_cRnl7_DK-7e640d2b12.png)

3、移除成功，聊天会话回到默认分组，个人回联系人、群组聊天回群组，社群聊天回社群

![](../../site/assets/images/_hRRzEMTJU-b3037e7435.png)

**删除分组**

1、点击删除分组按钮

![](../../site/assets/images/n_CvwO1hp--03bbcce1cc.png)

2、弹出确认删除界面，点击删除按钮（界面提示：此分组中的所有聊天将自动移至默认分组。）

![](../../site/assets/images/ebXnICTEfx-dfe3619b42.png)

**三、分组转移**

**在新分组里面添加聊天选择要转移的会话，就可以转移到新分组**
