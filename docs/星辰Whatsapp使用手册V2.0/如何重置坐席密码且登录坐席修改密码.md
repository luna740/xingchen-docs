# 如何重置坐席密码且登录坐席修改密码

分类：星辰Whatsapp使用手册V2.0
更新时间：2026-05-20T21:58:16+08:00
ID：7d0b620f82cc816acc54d963

**本文说明管理员如何重置坐席密码，以及坐席登录后如何自行修改密码。**

> 注意：管理员重置后的新密码只会在弹窗中显示一次，请及时复制并发送给对应坐席。

## 一、管理员重置坐席密码

1. 在左侧菜单进入【客服坐席】。
2. 打开【坐席列表】，找到需要重置密码的坐席账号。
3. 点击该账号右侧的【重置密码】。

   ![](../../site/assets/images/lm7x-0GI7i-0e031829ee.png)

4. 系统弹出确认窗口后，点击【确认】。

   ![](../../site/assets/images/vkqUHMZJMj-a848c491f7.png)

5. 页面提示密码重置成功后，复制弹窗中的新密码。
6. 将新密码发送给对应坐席，用于后续登录。

   ![](../../site/assets/images/EhEUTQU6yf-09ef3b8056.png)

## 二、坐席登录后修改密码

坐席拿到管理员重置后的新密码后，可以登录坐席系统并自行修改为常用密码。

1. 使用当前坐席账号和新密码登录坐席系统。
2. 登录成功后，点击右上角账号下拉按钮。
3. 点击【修改密码】，打开修改密码窗口。

   ![](../../site/assets/images/1Vj9GJxSyV-9fd85f590c.png)

4. 输入旧密码。
5. 输入新的坐席密码，并再次确认。
6. 点击【修改】，完成密码更新。

   ![](../../site/assets/images/Xwibvo6HN9-78f0c8c468.png)

> 提示：修改完成后，请使用新密码重新登录确认，避免后续因密码记录错误影响使用。
