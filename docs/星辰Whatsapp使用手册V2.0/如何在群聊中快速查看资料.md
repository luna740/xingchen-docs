# 如何在群聊中快速查看资料

分类：星辰Whatsapp使用手册V2.0
更新时间：2026-05-20T20:40:44+08:00
ID：d4e55cd8ffc5569b71db6eeb

**本文说明如何在群聊中快速查看成员资料。适合需要确认群成员身份、查看资料信息或进行后续沟通前核对信息的场景。**

## 一、进入群聊

1. 打开需要查看成员资料的群聊。
2. 在群聊消息或成员区域找到目标成员。

## 二、快速查看成员资料

1. 点击目标群成员的头像。
2. 系统会打开该成员的资料信息。

   ![](../../site/assets/images/IoRWoX95e0-52199ce356.png)
   ![](../../site/assets/images/T9sTiUIRAA-eddf602889.png)

> 提示：通过头像查看资料可以快速核对成员信息，不需要先添加好友。
