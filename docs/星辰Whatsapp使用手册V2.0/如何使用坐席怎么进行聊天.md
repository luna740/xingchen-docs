更新时间：2026-03-09T08:16:54.384Z

注：在后台点击进入坐席只能查看，不能聊天

**登录座席WhatsApp账号**

访问链接：星辰坐席系统

![](../../site/assets/images/wdFQn6PTKs-31120cf441.png)

输入正确的账号、密码、验证码后，点击【确定】按钮

![](../../site/assets/images/ndI9Xsp4R6-4781beeb5c.png)

登录成功后进入坐席系统主页面(操作跟WhatsApp网页端一致的)
