# 关于信任盾牌标识

分类：星辰whatsapp协议优势
更新时间：2026-03-20T08:39:08.828Z

## 1、为什么做信任盾牌标识：

有信任盾牌账号发信息不会触发风控，方便用户辨别对方账号是否可以主动发信息
只要粉丝主动发消息或者回复你的消息，就会获取盾牌标识

所有星辰协议的盾牌标识是共享的，假设账号A取得粉丝1的信任盾牌标识，那么账号B或者新注册的账号C一样可以直接向粉丝1发聊天并不会触发风控

## 3、通过群组和社群，群成员列表，查看成员，点击发信息按钮，到聊天会话可以看到是否信任

![](../../site/assets/images/uJ5ty8kwJt-4b8d2f926e.png)

**接粉收发消息本身没有风控，加好友这个我们统计到这个版本会有封号风险，不加好友不影响任何功能和风控，星辰建议不要点击加好友按钮**

![](../../site/assets/images/2fHd-fpX-c-c7415006a6.png)
